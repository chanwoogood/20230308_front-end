CSS 텍스트 관련 속성

font-family

글씨체를 지정한다. 없으면 순서에 따라 다음 폰트가 적용된다. font-family는 글씨체를 지정한다.
글씨체를 여러개 지정할 수 있는데, 첫번째 폰트가 없으면 두번째, 두번쨰 폰트가 없으면 세번쨰 이런 식으로 적용된다. 폰트 이름의 띄어쓰기가 있다면 " "사이에 작성한다.

font-size는 글씨의 크기를 지정한다.
px, cm, 키워드(예약어)와 같은 절대 크기 혹은 rem, em, %와 같은 상대 크기로 지정이 가능하다.

font-style

글씨의 스타일을 변경한다.

font-weight

폰트의 가중도(굵기)를 지정한다. normal, bold와 같은 키워드 혹은 100~900(100단위로) 사이의 숫자를 지정한다.
폰트 자체에 해당 굵기를 가지고 있어야 동작한다.

color

글씨 색을 지정한다.
키워드 혹은 색상코드(Hex, rgb)를 지정한다.

text-align

text-align은 글자의 정렬방향을 지정한다
인라인 요소(컨텐츠)가 아니라 글씨를 감싸고 있는 블록 요소에 지정해야한다.

line-height

줄의 높이를 지정한다.

text-decoration

글씨에 줄을 긋는다.

text-shadow

글자에 그림자 효과를 준다.

text-transform

글자를 대문자 혹은 소문자로 변환한다. Lorem ipsum dolor sit amet.

letter-spacing / word-spacing

letter-spacing 글자간의 간격(자간)을 조절한다.
word-spacing은 단어간의 간격을 조절한다. 띄어쓰기가 기준이다.
