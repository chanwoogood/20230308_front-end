블록 레벨 요소(Bock Level Element)

블록 레벨 요소는 영역을 지정할 때 사용한다.

블록 요소는 너비와 높이가 지정이 가능하다.

블록 요소의 너비는 기본적으로 부모 요소의 너비만큼 늘어난다.(기본값 auto)

블록 요소의 높이는 기본적으로 자신이 가진 컨텐츠만큼만 늘어난다.(기본값 auto)

블록 요소는 너비와 상관없이 줄바꿈을 일으킨다

인라인 레벨 요소(Inline Level Element)

인라인 레벨 요소는 특정 텍스트를 지정할 때 사용한다.

인라인 요소는 너비와 높이 지정이 불가능하다.

인라인 요소는 자신이 가진 컨텐츠만큼의 너비와 높이를 가진다.

인라인 요소는 줄바꿈을 일으키지 않는다.

width(너비) / height(높이)

너비와 높이에 %를 사용하면 부모 너비/높이에 대한 백분위로 지정된다.
